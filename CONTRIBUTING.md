# Contributing

Contributions are welcome! Here's how to help:

1. **Found a bug?** Open an issue with the script that broke it
2. **New deobfuscation technique?** Open a PR to `deobfuscator/static/<language>.py`
3. **Want a feature?** Open an issue first to discuss it

## Development

```bash
git clone git@github.com:gnaixnaij/deobfuscator-web.git
cd deobfuscator-web
pip install -r requirements.txt
python app.py
```
